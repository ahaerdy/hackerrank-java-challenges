# ☕ Desafios de Programação em Java

Repositório dedicado à resolução dos desafios de Java da plataforma [HackerRank](https://www.hackerrank.com/domains/java), com soluções comentadas e organizadas por categoria.

---

## 📌 Sobre este repositório

Este repositório tem como objetivo documentar a jornada de aprendizado e prática em Java por meio dos desafios propostos pelo HackerRank. O roteiro segue a trilha oficial da plataforma, cobrindo desde conceitos introdutórios até tópicos avançados como padrões de projeto, expressões lambda e criptografia.

Cada desafio possui sua própria subpágina contendo:

- Descrição do problema (traduzida para o português)
- Análise da solução adotada
- Código-fonte comentado em Java
- Informações sobre dificuldade e pontuação

---
